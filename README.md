# com.faber04.demo.pwa

This repository hosts a **simple Progressive Web App (PWA) demo** focused on the essentials.

## Purpose

Build a minimal, practical PWA example that demonstrates core capabilities:

- App installability
- Offline support (basic caching strategy)
- Fast loading and lightweight UX

## Tech stack (planned)

The app will be built with:

- **TypeScript**
- **HTML**
- **CSS**

Libraries or frameworks may be introduced later, but they are not fixed yet.
